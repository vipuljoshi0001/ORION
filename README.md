# ORION — Aptitude Mastery Platform

## Overview

**ORION** is a focused learning platform designed to help users master **aptitude and reasoning skills**, which form the foundation of most hiring processes.

In today’s competitive environment, the first elimination round in almost every company — including top tech firms like Google — begins with **aptitude and logical reasoning**, followed by DSA and coding rounds.

Despite its importance, students often lack a structured way to consistently practice and track their progress.

**ORION solves this problem.**

It provides a streamlined environment where users can:

* Practice daily aptitude problems
* Track their performance
* Build consistency through streaks
* Receive instant feedback and scoring

---

##  Problem Statement

Most students:

* Focus heavily on coding (DSA)
* Ignore aptitude preparation
* Lack consistency and performance tracking

This leads to failure in **initial screening rounds**, even before reaching technical interviews.

---

##  Our Solution

ORION introduces a **daily problem-solving ecosystem** where:

* Users log in securely
* Solve curated aptitude problems
* Get instant correctness + scoring feedback
* Track progress through analytics and heatmaps

This ensures **continuous improvement and exam readiness**.

## ⚙️ System Workflow

```
User Login/Register
        ↓
Fetch Daily Problem
        ↓
User Solves Question
        ↓
Submit Answer
        ↓
Backend Evaluation
        ↓
Return Result (Correct/Wrong + Score)
        ↓
Update Progress & Dashboard
```
